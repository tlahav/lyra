# Installing Lyra

Lyra can be installed using npm, yarn, or pnpm as follows:

npm i @nearform/lyra

yarn add @nearform/lyra

pnpm add @nearform/lyra

Once Lyra is installed, you can easily import in any project. Lyra exposes its own types, ESM modules, and CJS modules.

Once you're ready, you can start creating a database.
